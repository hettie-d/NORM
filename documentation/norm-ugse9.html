9 Known Issues

‘key names

The JSON keys are case-sensitive but are derived from PostgreSQL-type attribute names that are case-insensitive. As a result, the key in the outgoing JSON objects may differ from those specified in the contract. The fix is easy, but the repository owner hates double-quoted names in SQL.

NORM_GEN

The schema name NORM_GEN is hard-coded.
